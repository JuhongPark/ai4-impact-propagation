## How to Use This Catalog

- **Looking for a specific paper?** Use Ctrl+F on the title or first author.
- **Looking for what's been written about a method?** Find the method section above, then check the "Discussed in" column.
- **Looking for a method to apply to a new problem?** Scan the domain sections and pick papers with matching conditions (data type, scale, interpretability needs).
- **Building a reading list?** Start with foundational theory, then move to method-specific papers in the relevant domain.

See also `INDEX.md` at the repository root for topic-based document navigation.
